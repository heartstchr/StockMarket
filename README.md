Steps to follow


###to clone git repository
**git clone**

###To get all dependencies of node
**npm install**

###To get all dependencies releted to frontend
**bower install**

###To run server
**forever -w server.js**

###To veiw in browser
**http://localhost:7777**



##**_Optimisation_**
	* Minified css.
	* css at the top and js at bottom.
	* No use of Bootstrap to reduce load time.
	* Reduce DNS lookup.
