#Steps to follow


*###To clone git repository
	**git clone**

*###To get all dependencies of server
	**npm install**

*###To get all dependencies of client
	**bower install**

*###To run server
	**forever -w server.js**

*###To veiw in browser
	**http://localhost:7777**



##**_Optimisation_**
	* Minified css.
	* css at the top and js at bottom.
	* No use of Bootstrap to reduce load time.
	* Reduce DNS lookup.
